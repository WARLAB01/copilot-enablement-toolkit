# Microsoft 365 Copilot Enablement Toolkit

An interactive prompt library and enablement platform for Microsoft 365 Copilot, featuring 157 curated prompts across five departments — HR, Corporate Real Estate, Marketing, Procurement, and a Universal Toolkit — with filtering, personalization, and workflow assembly tools.

**Live site:** [warlab01.github.io/copilot-enablement-toolkit](https://warlab01.github.io/copilot-enablement-toolkit/)

## Features

- **Prompt Library** — 157 prompts filterable by department, Copilot surface, environment, and category
- **Prompt Personalizer** — Fill placeholders with your own context before copying
- **Workflow Assembler** — Chain prompts into multi-step workflows by goal
- **Adoption Tracker** — Monitor rollout progress across departments
- **Quick Reference** — Surface-specific cheat sheets for Teams, Outlook, Word, Excel, PowerPoint, and more

## Data

Prompt data is maintained in `copilot-prompt-library.csv` and embedded directly in the single-file app for zero-dependency deployment.
